シンプルなアニメーション

transition-property: / transition-duration:

要素に動きを持たせる際に指定するプロパティ。
transition-propertyでどの値に対して変化をもたらすか、
transition-durationで変化までにどれだけの時間をかけるかを指定する。
例)boxプロパティがマウスホバーした際に、1秒かけて丸くなるようにしたい場合、

①擬似クラスを用いてホバー時に丸くなるように指定
.box:hover{ border-radius:50% }

→

②.boxクラスにtransition-property/transition-durationを指定
.box{ transition-property:border-radius; transition-duration: 1s; }

→

※.boxではなく:hoverクラスに
transition-property/transition-durationを指定した場合、
"マウスホバー中"にしか適用されず
ホバーを外した瞬間にパッと切り替わってしまう
:hover{ transition-property:border-radius; transition-duration: 1s; }

→

transform / translate / rotate / scale

transiton-propertyに指定するプロパティとして、要素を変形させるtransformがよく使われる。
transformには、移動させるtranslate、回転させるrotate、拡大・縮小させるscaleなどさまざまな値を指定できる。
なお、transformプロパティは、インラインボックスに対しては無効となる。

translate

右に40px、上に40px動かす場合
.box:hover{ transform:translate(40px, -40px); }
（X軸、Y軸どちらかのみを指定する場合translateX() / translateY()でも指定可能）

→

rotate

時計回りに45度傾ける場合
.box:hover{ transform: rotate(45deg); }
（反時計回りに傾けたい場合はrotate(-45deg)と指定する）

→

scale

X軸を2倍、Y軸を半分にする場合
.box:hover{ transform: scale(2, .5); }
（X軸、Y軸どちらかのみを指定する場合scaleX() / scaleY()でも指定でき、
XYを同じ比率で拡大、縮小させたい場合はscale(2) / sccale(.5)で可能）

→

transform-origin

上のrotate、scaleでは、デフォルトでは要素の中心が基点となるが、transform-originを使うことで基点を変更できる

"左上を基点に"時計回りに45度傾ける場合
.box{ transform-origin:top left; }
.box:hover{ transform: rotate(45deg); }

→

"右下を基点に"X軸を2倍、Y軸を半分にする場合
.box{ transform-origin:bottom right; }
.box:hover{ transform: scale(2, .5); }

→

変形の組み合わせ

transformの値は半角空白で区切って値を指定することで、変形を組み合わせることができる。
なお、指定した値は"右から左"の順で適用される
例)transform:translate(100px) rotate(45deg);

transform:translate(100px) rotate(45deg);
「時計回りに45度傾いたあと、右に100px移動」する

→

逆に配置した場合、(transform:rotate(45deg) translate(100px);)
「右に100px移動したあと、時計回りに45度傾く」
なお、この時”trannsform-originの位置は初期値から変わらない”ため、
このように変形する

→

transition-delay

アニメーションの開始を遅らせるためのプロパティ。

.box{ transition-delay:1s; }
キッカケ（今回マウスホバー）から1秒後にアニメーションが開始される
ホバーを離してからも同様に1秒後に戻り始める

→

.box:hover{ transition-delay:1s; }
:hoverに指定した場合、ホバーを離した時にはかからずすぐに戻る

→

1秒以下(例えば0.5秒)で指定したい場合、.5sと指定してもよいが、
msを用いて500msとしても同じ値をとる
.box{ transition-delay:.5s; }＝.box{ transition-delay:500ms; }

→

transition-timing-function

transitonプロパティの動きに対して変化をつけたい場合に使用するプロパティ

transition-timing-function: ease;

デフォルトの動き
速度がゆっくり上がって、ゆっくり落ちる

→

~: ease-out;

動き始めがパッと弾かれたようになる。
小さな(軽い)UI部品で用いられる

→

~: ease-in-out;

easeより動き始めが遅い
大きな(重たい)UI部品で用いられる

→

~: linear;

初めから終わりまで等速で動く
ローディングアイコンなどで用いられる

→

transitionの一括指定プロパティ

transition関連のプロパティを指定する際、一括指定で設定することができる
注意点として、transition-duration、transition-delayと秒数で指定する2つのプロパティをまとめる場合、
最初がtransition-duration、次がtransition-delayの順番で指定される。
その順番以外は順不同でOK。

例)

.box{
transition-property: transform;
transition-duration: 2s;
transition-timing-function: ease-out;
}

＝

.box{
transition: transform 2s ease-out;

複雑なアニメーション

animation-name / animation-duration / animation-iteration-count

taransitionのような、キッカケが必要で単調な動きのアニメーションに対し、animationを使うことで、
より複雑なアニメーションを作ることができる。

animation-nameで定義したいアニメーション名（自由に設定可能）を指定し、
animation-durationでアニメーションの１サイクルにどれだけ時間をかけるか、
animation-iteration-countで、アニメーションを何回繰り返すか　をそれぞれ指定する。

@keyframes

animation-name: move-around;と指定した際、このmove-aroundを定義するためには、@keyframesを使う。

アニメーションを指定したい要素とは別で@keyframes move-around { }という形を作り、
中にアニメーションの定義を指定していく。
この中には、アニメーションの1サイクルを100%とし、何％時点でどう変わるか、という形で指定する。

.box{
width: 80px;
height: 80px;
background: pink;

animation-name:

move-around

; }

animation-duration: 4s;

animation-iteration-count: infinite;

@keyframes

move-around

{

25%{ transform: translateX(100px); }

50%{ transform: translate(100px , 50px); }

75%{ transform: translateY(50px); }

100%{ transform: translate(0 , 0); }

}

animation-delay

アニメーションの開始を遅らせるプロパティ。

.box{
width: 80px;
height: 80px;
background: pink;
animation-name: move-around;
animation-duration: 4s;
animation-iteration-count: infinite;

animation-delay:

1s

; }

@keyframes move-around{

25%{ transform: translateX(100px); }

50%{ transform: translate(100px , 50px); }

75%{ transform: translateY(50px); }

100%{ transform: translate(0 , 0); }

}

animation-direction

アニメーションのサイクルの仕方を指定するプロパティ。
animation-direction: alterate;を指定すると、100%まで行った段階で逆再生し0%に戻る動きをとる

.box{
width: 80px;
height: 80px;
background: pink;
animation-name: move-around;
animation-duration: 4s;
animation-iteration-count: infinite;

animation-direction:

alternate

; }

@keyframes move-around{

25%{ transform: translateX(100px); }

50%{ transform: translate(100px , 50px); }

75%{ transform: translateY(50px); }

100%{ transform: translate(0 , 0); }

}

animationの一括指定プロパティ

animationもこれまで同様、一括指定プロパティで指定することが可能。
transition同様に、時間を指定する値が2つ出てきた場合は、最初がduration、次がdelayの値となる。

.box{
width: 80px;
height: 80px;
background: pink;
animation-name: move-around;
animation-duration: 4s;
animation-iteration-count: infinite;
animation-delay: 1s;
animation-direction: alternate;

→

.box{
width: 80px;
height: 80px;
background: pink;
animation: move-around 4s infinite 1s alternate;

実用的なアニメーション

buttonのスタイリング

ボタンにのプロパティに対し、スタイリングを整え、
ホバーした際に色がふわっと反転するように指定する

button{
all: unset;
border:2px solid black;
font-weight: bold;
border-radius: 4px;
padding: 4px 8px;
cursor:pointer;
transition: background .3s; }

button:hover{
background:black;
colir:white;
}

ローディングアイコン

手順
①<div class>でclass指定した素体をHTMLで書く
②CSSでclassに対し基本的なスタイル付け（サイズ・ボーダー）を指定
③ainmationの一括指定プロパティで名前(spin)指定、duration、iteration-countを指定
④@keyframes spinを書き、transform:rotateで「100%時に360deg回転する」指定を行う

<div class="loading"></div>

.loading{
width: 32px;
height:32px;
border-top:8px solid red;
border-right:8px solid blue;
border-bottom:8px solid green;
border-left:8px solid yellow;
border-radius: 50%;
animation:spin 1s infinite;
}

@keyframes spin{
100%{
transform:rotate(360deg);
}

→

⑤ここまでだと上のようにanimationのデフォルトの動きになるため、
ローディングアイコンらしく等速で動かすためtransition-timing-functionと同じように
animationn-timing-functionのlinearを一括指定プロパティの値に入れる。

<div class="loading"></div>

.loading{
width: 32px;
height:32px;
border-top:8px solid red;
border-right:8px solid blue;
border-bottom:8px solid green;
border-left:8px solid yellow;
border-radius: 50%;
animation:spin 1s infinite linear;
}

@keyframes spin{
100%{
transform:rotate(360deg);
}

→

ポップアップメッセージ

手順
①ローディングアイコン同様に、<div class>でclass指定したメッセージの素体をHTMLで書き
　CSSでclassに対し基本的なスタイル付け（幅・ボーダー）を指定
②今回要素の場所を固定するためにpositionプロパティを使用し右下に固定する
③ainmationの一括指定プロパティで名前(popup)とduration指定、今回iteration-countは初期値(1回)のため省略
④@keyframes popupを書き、transform:translateXで「0%・100%時には外に、20%~80%で初期値にでてくる」指定を行う

<div class="message">
hallo!!</div>

.message{
width: 300px;
background:black;
color:white;
padding: 8px 16px;
border-radius: 4px;
position:absolute;
right:32px;
bottom: 32px;
animation:popup 3s;
}

@keyframes popup{
0%{ transform:translateX(100px); }
20%{ transform:translateX(0); }
80%{ transform:translateX(0); }
100%{ transform:translateX(100px); }

→

Hallo!!

⑤この状態では、上記のように100%まで完了したあと初期値に戻る動きをするため、
animation-fill-mode:のプロパティで完了後の状態を指定、
forwards;の値で「完了後の状態をキープ」することが可能。
なお、このプロパティも一括指定に含めることができるため、以下のように記入。

<div class="message">
hallo!!</div>

.message{
width: 300px;
background:black;
color:white;
padding: 8px 16px;
border-radius: 4px;
position:absolute;
right:32px;
bottom: 32px;
animation:popup 3s forwards;
}

@keyframes popup{
0%{ transform:translateX(100px); }
20%{ transform:translateX(0); }
80%{ transform:translateX(0); }
100%{ transform:translateX(100px); }

→

Hallo!!

⑥さらに、不透明度を上げながらふわっと出現させたいため、
@keyframesにopacityプロパティを使用する。
※display:none;で非表示にして再表示という発想もでるが、
displayプロパティはtransitionやanimationに対応していないため使用不可。

<div class="message">
hallo!!</div>

.message{
width: 300px;
background:black;
color:white;
padding: 8px 16px;
border-radius: 4px;
position:absolute;
right:32px;
bottom: 32px;

}

@keyframes popup{
0%{ transform:translateX(100px);
opacity: 0; }
20%{ transform:translateX(0);
opacity: 1; }
80%{ transform:translateX(0);
opacity: 1; }
100%{ transform:translateX(100px);
opacity: 0; }

→

Hallo!!

なお、ポップアップ下にボタンがある場合、ポップアップに阻害され下部のボタンが押せなくなることがある。
その場合は、ポップアップの要素にpointer-eventsというプロパティを@keyframesの100%につけ、値をnoneとすることで
指定した要素が100%時点でマウスイベントの対象外となり無視されるようになる。
